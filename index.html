Welcome to FizzBuzz, the early programming challenge every noob is completing.

In this code I am displaying the results of the classic FizzBuzz challenge with HTML and CSS (via Bootstrap 4).

This is my first attempt to display the results of FizzBuzz onscreen, outside of the console.

By clicking the button below you will see the following:

  • If a number is divisible by 3 with no remainder, the word "Fizz"
  • If a number is divisible by 5 with no remainder, the word "Buzz"
  • If a number is divisible by 3 & 5 with no remainder, the word "FizzBuzz"
  • If a number is not evenly divisible by 3 or 5, the number will appear
